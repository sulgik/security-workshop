Gemini API 무료 키 발급 및 Colab 설정

Google 계정만 있으면 무료 · 별도 결제 불필요

1

Google 계정 로그인

Gmail 계정으로 aistudio.google.com에 접속합니다.

2

API 키 발급

좌측 메뉴 Get API key → API key 만들기 클릭 후 키를 복사합니다.
⚠ 키는 절대 타인에게 공유 금지

3

.env 파일 생성

메모장에 아래 내용을 작성 후 .env_D3_L1으로 저장합니다.

GEMINI_API_KEY=여기에_복사한_키_붙여넣기
4

Colab에 업로드

노트북 열기 → 왼쪽 📁 파일 아이콘 → .env 파일 드래그 앤 드롭 업로드
